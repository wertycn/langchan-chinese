# 介绍

**LangChain** 是一个用于开发由语言模型驱动的应用程序的框架。它使得应用程序具备以下特点：
- **数据感知性**：将语言模型连接到其他数据源
- **主动性**：允许语言模型与其环境进行交互

LangChain 的主要价值主张包括：
1. **组件**：用于处理语言模型的抽象，以及每个抽象的一系列实现。这些组件是模块化的，无论您是否使用LangChain框架的其他部分，都可以方便地使用。
2. **现成的链**：用于完成特定高级任务的组件结构组合。

现成的链条使得入门变得容易。对于更复杂的应用和细致的用例，组件使得定制现有链条或构建新链条变得容易。

## 模块

LangChain为以下模块提供了标准的、可扩展的接口和外部集成，按复杂程度从低到高列出：

#### [模型 I/O](/docs/modules/model_io/)
与语言模型进行接口交互
#### [数据连接](/docs/modules/data_connection/)
与特定应用数据进行交互
#### [链](/docs/modules/chains/)
构建调用序列
#### [代理](/docs/modules/agents/)
根据高级指令让链选择使用哪些工具
#### [内存](/docs/modules/memory/)
在链的运行之间保持应用程序状态
#### [回调](/docs/modules/callbacks/)
记录和流式传输任何链的中间步骤
