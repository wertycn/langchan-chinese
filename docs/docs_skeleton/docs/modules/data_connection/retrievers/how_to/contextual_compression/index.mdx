# 上下文压缩

检索中的一个挑战是，通常您不知道在将数据导入到系统时，文档存储系统将面临哪些特定的查询。这意味着与查询最相关的信息可能埋藏在具有大量无关文本的文档中。将完整的文档通过应用程序传递可能导致更昂贵的LLM调用和较差的响应。

上下文压缩的目的是解决这个问题。其思想很简单：不是立即将检索到的文档原样返回，而是可以利用给定查询的上下文对其进行压缩，以便只返回相关的信息。“压缩”在这里既指对单个文档内容进行压缩，也指对整个文档进行过滤。

要使用上下文压缩检索器，您需要：
- 一个基础检索器
- 一个文档压缩器

上下文压缩检索器将查询传递给基本的检索器，并将初始文档传递给文档压缩器。文档压缩器通过减少文档的内容或完全删除文档来缩短文档列表。
