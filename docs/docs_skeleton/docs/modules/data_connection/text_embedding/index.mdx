# 文本嵌入模型

Embeddings类是一个与文本嵌入模型进行交互的类。有很多嵌入模型的提供者（OpenAI、Cohere、Hugging Face等），这个类被设计为为所有提供者提供一个标准化的接口。

嵌入层创建文本的向量表示。这很有用，因为这意味着我们可以在向量空间中思考文本，并进行语义搜索，即在向量空间中寻找最相似的文本。

LangChain中的基本Embeddings类公开了两个方法：一个用于嵌入文档，另一个用于嵌入查询。前者以多个文本作为输入，而后者以单个文本作为输入。之所以将它们作为两个单独的方法是因为一些嵌入提供者对于文档（用于搜索）和查询（搜索查询本身）有不同的嵌入方法。
