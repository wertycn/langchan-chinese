# 向量存储

存储和搜索非结构化数据最常见的方法之一是对其进行嵌入并存储生成的嵌入向量，然后在查询时对非结构化查询进行嵌入，并检索与嵌入查询最相似的嵌入向量。向量存储负责存储嵌入数据并为您执行向量搜索。

## 入门指南

此步骤演示了与VectorStores相关的基本功能。与向量存储一起工作的关键部分是创建要放入其中的向量，通常是通过嵌入来创建的。因此，在深入学习之前，建议您先熟悉[text embedding model](/docs/modules/model_io/models/embeddings.html)接口。
