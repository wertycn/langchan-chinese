# 数据连接

许多LLM应用程序需要用户特定的数据，这些数据不是模型的训练集的一部分。LangChain为您提供了加载、转换和查询数据的构建模块，您可以通过以下方式使用它们：

- [文档加载器](/docs/modules/data_connection/document_loaders/): 从许多不同的源加载文档
- [文档转换器](/docs/modules/data_connection/document_transformers/): 拆分文档、删除冗余文档等
- [文本嵌入模型](/docs/modules/data_connection/text_embedding/): 将非结构化文本转换为浮点数列表
- [向量存储](/docs/modules/data_connection/vectorstores/): 存储和检索嵌入数据
- [检索器](/docs/modules/data_connection/retrievers/): 查询您的数据
