# 文档转换器

一旦您加载了文档，通常会希望对其进行转换，以更好地适应您的应用程序。最简单的例子是，您可能希望将长文档拆分为更小的块，以适应模型的上下文窗口。LangChain具有许多内置的文档转换器，可以轻松地拆分、合并、过滤和其他方式操作文档。

## 文本拆分器

当您需要处理较长的文本时，有必要将文本拆分成块。
尽管这听起来很简单，但在这里存在很多潜在的复杂性。理想情况下，您希望将语义相关的文本片段保持在一起。"语义相关"的含义可能取决于文本的类型。
本笔记本展示了几种实现这一目标的方法。

在高层次上，文本拆分器的工作方式如下：

1. 将文本拆分成小而有意义的块（通常是句子）。
2. 将这些小块组合成更大的块，直到达到一定的大小（使用某个函数进行测量）。
3. 一旦达到了这个大小，将该块作为自己的文本片段，并开始创建一个新的文本块，其中包含一些重叠（以保持块之间的上下文）。

这意味着您可以沿两个不同的轴对文本分割器进行自定义：

1. 文本如何被分割
2. 块大小如何被测量
