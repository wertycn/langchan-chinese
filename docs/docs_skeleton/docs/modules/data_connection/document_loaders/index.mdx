# 文档加载器

使用文档加载器从源加载数据作为`Document`。`Document`是一段文本和相关的元数据。例如，有用于加载简单的`.txt`文件的文档加载器，用于加载任何网页的文本内容，甚至用于加载YouTube视频的转录。

文档加载器提供了一个"load"方法，用于从配置的源加载数据作为文档。它们可以选择性地
实现一个“延迟加载”，以便将数据懒加载到内存中。
