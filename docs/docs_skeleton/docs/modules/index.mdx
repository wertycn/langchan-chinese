# 模块

LangChain为以下模块提供了标准的、可扩展的接口和外部集成，按照从简单到复杂的顺序列出：

#### [模型输入输出](/docs/modules/model_io/)
与语言模型进行接口交互
#### [数据连接](/docs/modules/data_connection/)
与特定应用程序的数据进行接口交互
#### [调用链](/docs/modules/chains/)
构建调用序列
#### [代理](/docs/modules/agents/)
让链条根据高级指令选择使用哪些工具
#### [内存](/docs/modules/memory/)
在链条的运行之间保持应用程序状态
#### [回调](/docs/modules/callbacks/)
记录和流式传输任何链条的中间步骤
