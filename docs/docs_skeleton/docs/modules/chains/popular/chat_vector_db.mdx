# 会话检索问答（Conversational Retrieval QA）

会话检索问答链（ConversationalRetrievalQAChain）在检索问答链的基础上增加了聊天历史组件。

它首先将聊天历史（可以明确传递或从提供的记忆中检索）与问题合并为一个独立的问题，然后从检索器中查找相关文档，最后将这些文档和问题传递给问答链，返回一个回答。

要创建一个检索器，您需要一个存储器。在下面的示例中，我们将从向量存储器创建一个检索器，而向量存储器可以从嵌入创建。
