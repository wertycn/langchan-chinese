# 提示模板

语言模型以文本作为输入，这个文本通常被称为提示。
通常，这不仅仅是一个硬编码的字符串，而是一个模板、一些示例和用户输入的组合。
LangChain提供了几个类和函数，使构建和处理提示变得容易。

## 什么是提示模板？

一个提示模板是指生成提示的一种可复制的方式。它包含一个文本字符串（"模板"），可以从最终用户那里接收一组参数并生成提示。

一个提示模板可以包含：
- 对语言模型的指令，
- 一组少量示例，以帮助语言模型生成更好的回复，
- 对语言模型的问题。
