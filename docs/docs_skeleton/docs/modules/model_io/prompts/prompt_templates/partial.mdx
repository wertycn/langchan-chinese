# 部分提示模板

与其他方法类似，将提示模板“部分化”是有意义的 - 例如，传递所需值的子集，以创建一个只期望剩余子集值的新提示模板。

LangChain 有两种方式支持这种功能：
1. 使用字符串值进行部分格式化。
2. 使用返回字符串值的函数进行部分格式化。

这两种不同的方式支持不同的用例。在下面的示例中，我们将介绍这两种用例的动机以及如何在LangChain中实现。

```jsx
import Example from "@snippets/modules/model_io/prompts/prompt_templates/partial.mdx"

<Example/>
```
