# 模型输入输出

任何语言模型应用程序的核心要素是模型。LangChain提供了与任何语言模型进行交互的基本组件。

- [提示](/docs/modules/model_io/prompts/): 模板化、动态选择和管理模型输入
- [语言模型](/docs/modules/model_io/models/): 通过常用接口调用语言模型
- [输出解析器](/docs/modules/model_io/output_parsers/): 从模型输出中提取信息
