# 聊天模型

聊天模型是语言模型的一种变体。
虽然聊天模型在内部使用语言模型，但它们提供的接口有所不同。
与提供“输入文本，输出文本”的API不同，它们提供的接口以“聊天消息”作为输入和输出。

聊天模型的API还比较新，所以我们还在探索正确的抽象层次。

以下是提供的文档的各个部分：

- **操作指南**：核心功能的详细步骤说明，例如流式传输、创建聊天提示等。

- **集成**：如何使用不同的聊天模型提供商（OpenAI、Anthropic等）。
