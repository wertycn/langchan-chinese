# LLMs

大型语言模型（LLMs）是LangChain的核心组件。
LangChain不提供自己的LLMs，而是提供与许多不同的LLMs交互的标准接口。

要获取更详细的文档，请查看以下内容：

- **操作指南**：核心功能的操作步骤，如流式处理、异步处理等。

- **集成**：如何使用不同的LLM提供者（OpenAI、Anthropic等）。

## 入门指南

有很多LLM提供商（OpenAI，Cohere，Hugging Face等）- `LLM`类旨在为所有这些提供商提供标准接口。

在本教程中，我们将使用OpenAI LLM包装器，尽管所述功能对于所有LLM类型都是通用的。
