# 自定义 LLM 代理（带有 ChatModel）

本文档介绍了如何基于聊天模型创建自己的自定义代理。

一个 LLM 聊天代理由以下三个部分组成：

- PromptTemplate: 这是用于指导语言模型操作的提示模板
- ChatModel: 这是驱动代理的语言模型
- `stop` 序列：指示 LLM 在找到该字符串时停止生成
- OutputParser: 这个决定了如何解析LLMOutput为一个AgentAction或AgentFinish对象
