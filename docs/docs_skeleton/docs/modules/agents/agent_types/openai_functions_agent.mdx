# OpenAI函数

某些OpenAI模型（如gpt-3.5-turbo-0613和gpt-4-0613）已经经过微调，可以检测到何时应该调用函数，并回复应该传递给函数的输入。
在API调用中，您可以描述函数，并让模型智能地选择输出一个包含调用这些函数的参数的JSON对象。
OpenAI函数API的目标是比通用的文本补全或聊天API更可靠地返回有效和有用的函数调用。

OpenAI函数代理是为了与这些模型配合使用而设计的。
