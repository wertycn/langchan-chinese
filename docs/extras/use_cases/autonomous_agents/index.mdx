# 自主（长期运行）代理

自主代理是设计成更长期运行的代理。您可以为它们提供一个或多个长期目标，它们将独立地朝着这些目标执行。这些应用程序结合了工具使用和长期记忆。

目前，自主代理还处于实验阶段，并基于其他开源项目进行开发。通过在LangChain原语中实现这些开源项目，我们可以获得LangChain的好处。 
轻松切换和尝试多个LLM，使用不同的向量存储作为内存，使用LangChain的工具集。
