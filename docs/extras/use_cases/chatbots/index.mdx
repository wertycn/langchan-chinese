# 聊天机器人

由于语言模型擅长生成文本，因此它们非常适合用于创建聊天机器人。
除了基本提示/语言模型，聊天机器人还有一个重要的概念是`记忆`。
大多数基于聊天的应用程序依赖于记住之前交互中发生的事情，而`记忆`就是为了帮助处理这个问题。
