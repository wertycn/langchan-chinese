# 与API交互

许多数据和信息存储在API后面。
本页面涵盖了LangChain中用于与API交互的所有资源。

## 链条

如果您刚开始，并且有相对简单的API，您应该从链条开始。
链条是一系列预定步骤，因此它们非常适合初学者，因为它们能够提供更多的控制，并让您更好地理解正在发生的事情。

- [API链条](/docs/modules/chains/how_to/api.html)

## 代理

代理人更加复杂，涉及多个查询到LLM以了解要做什么。
代理人的缺点是您拥有的控制较少。好处是它们更强大，
这使您可以在更大更复杂的模式上使用它们。

- [OpenAPI代理人](/docs/modules/agents/tools/toolkits/examples/openapi.html)
