# Cassandra

>[Cassandra](https://en.wikipedia.org/wiki/Apache_Cassandra) 是一个免费开源的分布式宽列存储、NoSQL数据库管理系统，旨在处理大量数据并跨多个通用服务器进行存储，通过提供高可用性而没有单点故障来保证系统的稳定性。`Cassandra`支持跨多个数据中心的集群，采用异步无主复制，为所有客户端提供低延迟的操作。 
> `Cassandra` 的设计是将 `Amazon` 的分布式存储和复制技术与 `Google` 的 `Bigtable` 数据和存储引擎模型相结合。

## 安装和设置
