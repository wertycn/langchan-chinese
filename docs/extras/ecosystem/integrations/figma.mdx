# Figma

>[Figma](https://www.figma.com/) 是一个用于界面设计的协作web应用程序。

## 安装和设置

Figma API 需要一个 `access token`、`node_ids` 和一个 `file key`。

`file key` 可以从 URL 中获取。https://www.figma.com/file/{filekey}/sampleFilename

`Node IDs` 也可以在 URL 中找到。点击任何元素，查找 '?node-id={node_id}' 参数。

`访问令牌` [说明](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens)。
