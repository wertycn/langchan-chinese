# OpenAI

>[OpenAI](https://en.wikipedia.org/wiki/OpenAI) 是一个美国人工智能（AI）研究实验室，由非盈利组织 `OpenAI Incorporated` 和其盈利子公司 `OpenAI Limited Partnership` 组成。`OpenAI` 旨在推动和发展友好的人工智能，并进行AI研究。`OpenAI` 系统在 `Microsoft` 的基于 `Azure` 的超级计算平台上运行。

> [OpenAI API](https://platform.openai.com/docs/models)由一系列具有不同功能和价格点的多样化模型驱动。
> 
> [ChatGPT](https://chat.openai.com)是由`OpenAI`开发的人工智能（AI）聊天机器人。

## 安装和设置
- 使用以下命令安装Python SDK：
```bash
pip install openai
```
- 获取OpenAI的API密钥，并将其设置为环境变量（`OPENAI_API_KEY`）
- 如果您想使用OpenAI的分词器（仅适用于Python 3.9+），请安装它
```bash
pip install tiktoken
```


## LLM

```python
from langchain.llms import OpenAI
```

如果您正在使用托管在`Azure`上的模型，您应该使用不同的包装器。
```python
from langchain.llms import AzureOpenAI
```

要详细了解`Azure`包装器的使用步骤，请参阅[此笔记本](/docs/modules/model_io/models/llms/integrations/azure_openai_example.html)

## 分词器

您可以在多个地方使用`tiktoken`分词器。默认情况下，它用于计算OpenAI LLMs的标记数。

您还可以在拆分文档时使用它来计算标记数。 
```python
from langchain.text_splitter import CharacterTextSplitter
CharacterTextSplitter.from_tiktoken_encoder(...)
```
更详细的步骤请参考[此笔记本](/docs/modules/data_connection/document_transformers/text_splitters/tiktoken.html)
