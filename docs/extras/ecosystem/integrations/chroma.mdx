# Chroma

>[Chroma](https://docs.trychroma.com/getting-started) 是一个用于构建带有嵌入的人工智能应用程序的数据库。

## 安装和设置

```bash
pip install chromadb
```


## VectorStore

存在一个对Chroma矢量数据库的封装，允许您将其用作矢量存储，
无论是用于语义搜索还是示例选择。

```python
from langchain.vectorstores import Chroma
```

对于Chroma包装器的更详细的步骤，请参阅[此笔记本](/docs/modules/data_connection/vectorstores/integrations/chroma.html)
