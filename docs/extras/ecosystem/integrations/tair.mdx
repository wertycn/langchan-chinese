# Tair

本页面介绍如何在LangChain中使用Tair生态系统。

## 安装和设置

使用 `pip install tair` 安装Tair Python SDK。

## 包装器

### VectorStore

存在一个对TairVector的封装，允许您将其用作向量存储，
用于语义搜索或示例选择。

要导入此向量存储:

```python
from langchain.vectorstores import Tair
```

有关Tair包装器的更详细步骤，请参见[此笔记本](/docs/modules/data_connection/vectorstores/integrations/tair.html)。
