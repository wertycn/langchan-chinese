# LanceDB

本页面介绍了如何在LangChain中使用[LanceDB](https://github.com/lancedb/lancedb)。
它分为两个部分：安装和设置，以及特定的LanceDB封装的引用。

## 安装和设置

- 使用`pip install lancedb`安装Python SDK

## 封装器

### VectorStore

存在一个围绕LanceDB数据库的封装器，使您可以将其用作向量存储，
无论是用于语义搜索还是示例选择。

要导入此向量存储：

```python
from langchain.vectorstores import LanceDB
```

要详细了解LanceDB包装器的使用方法，请参阅[这个笔记本](/docs/modules/data_connection/vectorstores/integrations/lancedb.html)。
