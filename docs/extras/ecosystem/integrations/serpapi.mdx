# SerpAPI

本页面介绍了如何在LangChain中使用SerpAPI搜索API。
它分为两个部分：安装和设置，以及特定SerpAPI包装器的参考。

## 安装和设置
- 使用`pip install google-search-results`安装所需的软件包
- 获取SerpAPI的API密钥，并将其设置为环境变量(`SERPAPI_API_KEY`)

## 包装器

### 实用工具

存在一个SerpAPI实用工具，用于包装此API。要导入此实用工具:

```python
from langchain.utilities import SerpAPIWrapper
```

对于此包装器的更详细说明，请参见[此笔记本](/docs/modules/agents/tools/integrations/serpapi.html)。

### 工具

您还可以将此包装器作为工具轻松加载（与代理一起使用）。
您可以通过以下方式实现：
```python
from langchain.agents import load_tools
tools = load_tools(["serpapi"])
```

有关更多信息，请参阅[此页面](/docs/modules/agents/tools)
