# Pinecone

本页面介绍如何在LangChain中使用Pinecone生态系统。
它分为两个部分：安装和设置，以及特定Pinecone包装器的参考。

## 安装和设置
安装Python SDK：
```bash
pip install pinecone-client
```


## Vectorstore

存在一个围绕Pinecone索引的包装器，可以将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain.vectorstores import Pinecone
```

要获取更详细的Pinecone向量存储的操作步骤，请查看[这个笔记本](/docs/modules/data_connection/vectorstores/integrations/pinecone.html)。
