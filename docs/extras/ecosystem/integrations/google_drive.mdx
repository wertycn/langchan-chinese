# Google Drive

>[Google Drive](https://en.wikipedia.org/wiki/Google_Drive) 是由谷歌开发的文件存储和同步服务。

目前，仅支持 `Google Docs`。

## 安装和设置

首先，您需要安装一些Python包。

```bash
pip install google-api-python-client google-auth-httplib2 google-auth-oauthlib
```

## 文档加载器

请参阅[使用示例和授权说明](/docs/modules/data_connection/document_loaders/integrations/google_drive.html)。


```python
from langchain.document_loaders import GoogleDriveLoader
```
