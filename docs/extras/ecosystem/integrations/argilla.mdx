# Argilla

![Argilla - 针对LLMs的开源数据平台](https://argilla.io/og.png)

>[Argilla](https://argilla.io/) 是一个针对LLMs的开源数据策划平台。
> 使用Argilla，每个人都可以通过更快的数据策划，使用人工和机器反馈来构建强大的语言模型。
> 我们为MLOps周期的每个步骤提供支持，从数据标注到模型监控。

## 安装和设置

首先，您需要按照以下步骤安装 `argilla` Python包：

```bash
pip install argilla --upgrade
```

如果您已经运行了Argilla服务器，那么您就可以开始了；但如果您没有，请按照以下步骤安装它。

如果您没有，请参考[Argilla - 🚀 快速入门](https://docs.argilla.io/en/latest/getting_started/quickstart.html#Running-Argilla-Quickstart)在HuggingFace Spaces上、本地或服务器上部署Argilla。
