# 2Markdown

>[2markdown](https://2markdown.com/)服务可以将网站内容转换为结构化的Markdown文件。


## 安装和设置

我们需要`API密钥`。请查看[获取API密钥的说明](https://2markdown.com/login)。


## 文档加载器

请参阅[使用示例](/docs/modules/data_connection/document_loaders/integrations/tomarkdown.html)。

from langchain.document_loaders import ToMarkdownLoader
