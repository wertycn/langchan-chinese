# Milvus

本页面介绍了如何在LangChain中使用Milvus生态系统。
它分为两个部分：安装和设置，以及特定Milvus包装器的引用。

## 安装和设置
- 使用 `pip install pymilvus` 安装Python SDK

## 包装器

### VectorStore

存在一个围绕Milvus索引的包装器，允许您将其用作矢量存储，
无论是用于语义搜索还是示例选择。

要导入此矢量存储：
```python
from langchain.vectorstores import Milvus
```

有关Miluvs包装器的更详细演示，请参阅[此笔记本](/docs/modules/data_connection/vectorstores/integrations/milvus.html)。
