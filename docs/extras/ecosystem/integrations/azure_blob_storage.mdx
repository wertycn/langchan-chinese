# Azure Blob 存储

>[Azure Blob Storage](https://learn.microsoft.com/zh-cn/azure/storage/blobs/storage-blobs-introduction) 是微软云的对象存储解决方案。Blob 存储专为存储大量的非结构化数据进行了优化。非结构化数据是指不符合特定数据模型或定义的数据，例如文本或二进制数据。

>[Azure 文件](https://learn.microsoft.com/zh-cn/azure/storage/files/storage-files-introduction) 提供了完全托管的存储服务，用于存储和访问文件。
> 在云中，可以通过行业标准的Server Message Block (`SMB`)协议、Network File System (`NFS`)协议和`Azure Files REST API`访问的文件共享。`Azure Files`基于`Azure Blob Storage`。

`Azure Blob Storage`的设计用途包括：
- 直接向浏览器提供图像或文档。
- 存储文件以进行分布式访问。
- 流式传输视频和音频。
- 写入日志文件。
- 存储数据用于备份和恢复、灾难恢复和归档。
- 存储数据以供本地或托管在Azure上的服务进行分析。

## 安装和设置

```bash
pip install azure-storage-blob
```
