# Wolfram Alpha

>[WolframAlpha](https://en.wikipedia.org/wiki/WolframAlpha) 是由`Wolfram Research`开发的一个答案引擎。
> 它通过计算来自外部数据源的答案来回答事实性查询。

本页面介绍了如何在LangChain中使用`Wolfram Alpha API`。

## 安装和设置
- 安装所需的依赖项 
```bash
pip install wolframalpha
```
- 前往Wolfram Alpha并注册开发者账号[点击这里](https://developer.wolframalpha.com/)
- 创建一个应用并获取你的`APP ID`
- 将你的APP ID设置为环境变量`WOLFRAM_ALPHA_APPID`


## 包装类

### 实用工具

有一个名为WolframAlphaAPIWrapper的实用工具，可以包装这个API。导入这个实用工具的方法:

```python
from langchain.utilities.wolfram_alpha import WolframAlphaAPIWrapper
```

对于这个封装的更详细的说明，请参阅 [这个笔记本](/docs/modules/agents/tools/integrations/wolfram_alpha.html)。

### 工具

您还可以将这个封装作为一个工具轻松加载（用于与 Agent 一起使用）。
您可以通过以下方式实现：
```python
from langchain.agents import load_tools
tools = load_tools(["wolfram-alpha"])
```

有关更多信息，请参阅[此页面](/docs/modules/agents/tools/getting_started.md)。
