# Azure Cognitive Search

>[Azure Cognitive Search](https://learn.microsoft.com/en-us/azure/search/search-what-is-azure-search)（以前称为`Azure Search`）是一项云搜索服务，为开发人员提供基础设施、API和工具，用于在Web、移动和企业应用程序中构建丰富的搜索体验，用于私有、异构内容的搜索。

>搜索是任何向用户展示文本的应用程序的基础，常见的场景包括目录或文档搜索、在线零售应用程序或针对专有内容的数据探索。创建搜索服务时，您将使用以下功能：
>- 用于对包含用户拥有内容的搜索索引进行全文搜索的搜索引擎
>- 丰富的索引功能，包括词法分析和可选的人工智能增强，用于内容提取和转换
>- 丰富的查询语法，用于文本搜索、模糊搜索、自动完成、地理搜索等功能
>- 通过Azure SDK中的REST API和客户端库进行编程
>- Azure在数据层、机器学习层和AI（认知服务）层面上的集成
