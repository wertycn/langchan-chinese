# Yeager.ai

本页面介绍如何使用 [Yeager.ai](https://yeager.ai) 来生成 LangChain 工具和代理。

## 什么是 Yeager.ai？
Yeager.ai 是一个旨在简化创建 AI 代理和工具过程的生态系统。 

它具有yAgents，一个无代码的LangChain代理构建工具，使用户能够轻松构建、测试和部署AI解决方案。利用LangChain框架，yAgents可以与各种语言模型和资源进行无缝集成，适用于开发人员、研究人员和各种应用场景中的AI爱好者。

## yAgents
低代码生成代理，旨在帮助您轻松构建、原型设计和部署Langchain工具。

### 如何使用？
```
pip install yeagerai-agent
yeagerai-agent
```
访问 http://127.0.0.1:7860

这将安装必要的依赖项并在您的系统上设置yAgents。第一次运行后，yAgents将创建一个.env文件，您可以在其中输入您的OpenAI API密钥。您也可以直接从Gradio界面的"Settings"选项卡中进行相同操作。

`OPENAI_API_KEY=<your_openai_api_key_here>`

我们推荐使用GPT-4，但是如果问题足够简单，该工具也可以与GPT-3一起使用。

### 使用yAgents创建和执行工具
yAgents使创建和执行基于人工智能的工具变得简单。以下是该过程的简要概述：
1. 创建工具：要创建一个工具，向yAgents提供一个自然语言提示。提示应清楚地描述工具的目的和功能。例如：
`创建一个返回第n个质数的工具`

2. 将工具加载到工具包中：要将工具加载到yAgents中，只需向yAgents提供一个相应的命令即可。例如：
`加载刚刚创建的工具到您的工具包中`

3. 执行工具：要运行一个工具或代理程序，只需向yAgents提供一个命令，其中包括工具的名称和任何必需的参数。例如：
`生成第50个质数`

您可以在[这里](https://www.youtube.com/watch?v=KA5hCM3RaWE)观看它的工作视频。

随着您对yAgents的熟悉度提高，您可以创建更高级的工具和代理程序，以自动化您的工作并提高您的生产力。
