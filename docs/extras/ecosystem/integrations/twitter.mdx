# Twitter

>[Twitter](https://twitter.com/) 是一个在线社交媒体和社交网络服务。

## 安装和设置

```bash
pip install tweepy
```

我们必须使用`Twitter API`令牌初始化加载器，并且我们需要设置Twitter的`用户名`。
