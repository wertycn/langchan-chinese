# Bedrock

>[Amazon Bedrock](https://aws.amazon.com/bedrock/)是一个完全托管的服务，通过API提供来自领先的人工智能初创公司和亚马逊的FMs，因此您可以从各种FMs中选择最适合您的用例的模型。

## 安装和设置

```bash
pip install boto3
```

## LLM

查看[用法示例](/docs/modules/model_io/models/llms/integrations/bedrock.html)。

```python
from langchain import Bedrock
```

## 文本嵌入模型

请参考[使用示例](/docs/modules/data_connection/text_embedding/integrations/bedrock.html)。
```python
from langchain.embeddings import BedrockEmbeddings
```
