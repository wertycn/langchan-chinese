# Hacker News（黑客新闻）

>[Hacker News](https://en.wikipedia.org/wiki/Hacker_News)（有时缩写为`HN`）是一个关注计算机科学和创业的社交新闻网站。它由投资基金和创业孵化器`Y Combinator`运营。一般而言，可以提交的内容被定义为“满足人们的知识好奇心的任何事物”。

## 安装和设置

对于此网站，没有特殊的设置要求。
