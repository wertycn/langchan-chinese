# Metal

本页面介绍了如何在LangChain中使用Metal。

## 什么是Metal？

Metal是一个为生产环境构建的管理检索和存储平台。您可以将数据索引到Metal中，并在其上运行语义搜索和检索。

![Metal](/img/MetalDash.png)

## 快速入门

请从[创建一个Metal账户](https://app.getmetal.io/signup)开始。

然后，您可以轻松地利用`MetalRetriever`类来开始检索数据，进行语义搜索、提示上下文等操作。该类接受一个`Metal`实例和一个参数字典，用于传递给Metal API。

```python
from langchain.retrievers import MetalRetriever
from metal_sdk.metal import Metal


metal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");
retriever = MetalRetriever(metal, params={"limit": 2})

docs = retriever.get_relevant_documents("search term")
```
