# Zep

>[Zep](https://docs.getzep.com/) - 用于LLM应用程序的长期记忆存储。

>`Zep` 存储、汇总、嵌入、索引和丰富了对话式人工智能聊天历史，并通过简单、低延迟的API进行公开访问。
>- 长期记忆持久性，无论您的汇总策略如何，都可以访问历史消息。
>- 基于可配置的消息窗口对内存消息进行自动摘要。一系列摘要被存储，为未来的摘要策略提供灵活性。
>- 对记忆和摘要进行自动标记计数，实现对提示组装的更精细控制。
>- 支持记忆的向量搜索，消息在创建时自动嵌入。
>- 提供Python和JavaScript的SDK。

`Zep` [项目](https://github.com/getzep/zep) 

## 安装和设置

pip install zep_python
