# Jina

本页面介绍了如何在LangChain中使用Jina生态系统。
它分为两个部分：安装和设置，以及对特定的Jina包装器的引用。

## 安装和设置
- 使用 `pip install jina` 安装Python SDK。
- 从[这里](https://cloud.jina.ai/settings/tokens)获取Jina AI Cloud的身份验证令牌，并将其设置为环境变量(`JINA_AUTH_TOKEN`)。

## 包装器

### 嵌入

存在一个Jina嵌入包装器，您可以使用它来访问。 
```python
from langchain.embeddings import JinaEmbeddings
```
要获取更详细的教程，请查看[此笔记本](/docs/modules/data_connection/text_embedding/integrations/jina.html)。
