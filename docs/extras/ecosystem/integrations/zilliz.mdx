# Zilliz

>[Zilliz Cloud](https://zilliz.com/doc/quick_start)是一个完全托管在云端的`LF AI Milvus®`服务，


## 安装和设置

安装Python SDK:
```bash
pip install pymilvus
```

## Vectorstore（向量存储）

Zilliz索引的包装器使您可以将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain.vectorstores import Milvus
```

要了解更详细的Miluvs包装器操作步骤，请参阅[此笔记本](/docs/modules/data_connection/vectorstores/integrations/zilliz.html)。
