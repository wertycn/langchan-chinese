# Telegram

>[Telegram Messenger](https://web.telegram.org/a/) 是一种全球可访问的免费、跨平台、加密、基于云的集中式即时通讯服务。该应用程序还提供可选的端到端加密聊天和视频通话、VoIP、文件共享和其他多种功能。

## 安装和设置

请参阅[设置说明](/docs/modules/data_connection/document_loaders/integrations/telegram.html)。
