# Notion数据库

>[Notion](https://www.notion.so/)是一个协作平台，具有修改过的Markdown支持，集成了看板、任务、维基和数据库。它是一个集合了笔记、知识和数据管理以及项目和任务管理的多合一工作空间。

## 安装和设置

所有的指令都在下面的例子中。

## 文档加载器

我们有两种不同的加载器：`NotionDirectoryLoader`和`NotionDBLoader`。

请参阅[NotionDirectoryLoader的用法示例](/docs/modules/data_connection/document_loaders/integrations/notion.html)。


```python
from langchain.document_loaders import NotionDirectoryLoader
```

请参阅[NotionDBLoader的使用示例](/docs/modules/data_connection/document_loaders/integrations/notiondb.html)。
