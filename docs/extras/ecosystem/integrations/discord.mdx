# Discord

>[Discord](https://discord.com/) 是一个语音通话和即时消息社交平台。用户可以通过语音通话、视频通话、文本消息、媒体和文件在私人聊天或作为名为 "服务器" 的社群的一部分进行通信。服务器是持久聊天室和语音通道的集合，可以通过邀请链接访问。

## 安装和设置


```bash
pip install pandas
```

按照以下步骤下载您的 `Discord` 数据：

1. 进入您的**用户设置**
2. 然后进入**隐私与安全**
3. 转到**请求我的全部数据**并点击**请求数据**按钮

您可能需要等待30天才能收到您的数据。您将在您在Discord上注册的地址收到一封电子邮件。该邮件将有一个下载按钮，您可以使用该按钮下载您的个人Discord数据。
