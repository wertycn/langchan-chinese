# Psychic

>[Psychic](https://www.psychic.dev/) 是一个平台，通过OAuth与`Notion`、`Zendesk`、`Confluence`和`Google Drive`等SaaS工具集成，并将这些应用程序中的文档同步到您的SQL或向量数据库中。您可以将其视为非结构化数据的Plaid。

## 安装和设置

```bash
pip install psychicapi
```

Psychic很容易设置 - 您只需导入`react`库，并使用您从[Psychic控制面板](https://dashboard.psychic.dev/)获取的`Sidekick API`密钥进行配置。当您连接应用程序时，您可以从控制面板查看这些连接，并使用服务器端库检索数据。

1. 在[控制面板](https://dashboard.psychic.dev/)上创建一个账户。
2. 使用[react库](https://docs.psychic.dev/sidekick-link)将Psychic链接模态框添加到您的前端react应用程序中。您将使用此功能来连接SaaS应用程序。
3. 创建连接后，您可以按照[示例笔记本](/docs/modules/data_connection/document_loaders/integrations/psychic.html)中的说明使用`PsychicLoader`

## 与其他文档加载器相比的优势

1. **通用API：** 不需要为每个SaaS应用程序构建OAuth流程和学习API，您只需集成Psychic，利用我们的通用API来检索数据。
2. **数据同步：** 您客户的SaaS应用程序中的数据可能很快就会变得陈旧。通过Psychic，您可以配置Webhooks以每天或实时更新您的文档。
3. **简化的OAuth：** Psychic会全程处理OAuth，这样您就不需要花时间为每个集成创建OAuth客户端，保持访问令牌的新鲜，并处理OAuth重定向逻辑。
