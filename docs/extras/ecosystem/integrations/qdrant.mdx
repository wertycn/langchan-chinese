# Qdrant

本页面介绍如何在LangChain中使用Qdrant生态系统。
它分为两个部分：安装和设置，以及对特定Qdrant包装器的引用。

## 安装和设置
- 使用 `pip install qdrant-client` 安装Python SDK

## 包装器

### VectorStore

存在一个围绕Qdrant索引的包装器，允许您将其用作向量存储，
无论是用于语义搜索还是示例选择。

要导入此向量存储：
```python
from langchain.vectorstores import Qdrant
```

要详细了解Qdrant包装器的使用方法，请参阅[这个笔记本](/docs/modules/data_connection/vectorstores/integrations/qdrant.html)。
