# Llama.cpp

本页面介绍了如何在LangChain中使用[llama.cpp](https://github.com/ggerganov/llama.cpp)。
它分为两部分：安装和设置以及对特定的Llama-cpp封装的引用。

## 安装和设置
- 使用`pip install llama-cpp-python`安装Python包
- 在[支持的模型](https://github.com/ggerganov/llama.cpp#description)中下载其中之一，并根据[说明](https://github.com/ggerganov/llama.cpp)将其转换为llama.cpp格式

## 封装器

### LLM

存在一个LlamaCpp LLM封装器，您可以使用它来访问 
```python
from langchain.llms import LlamaCpp
```
对于更详细的步骤，请参阅[此笔记本](/docs/modules/model_io/models/llms/integrations/llamacpp.html)

### 嵌入

LlamaCpp提供了一个嵌入包装器，您可以通过以下方式访问 
```python
from langchain.embeddings import LlamaCppEmbeddings
```
有关更详细的步骤，请参阅[这个笔记本](/docs/modules/data_connection/text_embedding/integrations/llamacpp.html)。
