# Docugami

>[Docugami](https://docugami.com)将业务文件转换为文档XML知识图，生成表示整个文档的XML语义树的森林。这是一个丰富的表示，包括文档中各个块的语义和结构特征。

## 安装和设置


```bash
pip install lxml
```

## 文档加载器

请参阅[用法示例](/docs/modules/data_connection/document_loaders/integrations/docugami.html)。

```python
from langchain.document_loaders import DocugamiLoader
```
