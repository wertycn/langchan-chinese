# 深湖

本页面介绍了如何在LangChain中使用深湖生态系统。

## 为什么选择深湖？
- 不仅仅是一个（多模态）向量存储库。您可以使用数据集来微调自己的LLM模型。
- 不仅存储嵌入向量，还存储原始数据，并具有自动版本控制功能。
- 真正无服务器。不需要其他服务，可以与主要的云提供商（AWS S3、GCS等）一起使用。

## 封装器

### VectorStore

在Deep Lake周围存在一个包装器，它是一个用于深度学习应用程序的数据湖，让您可以将其用作向量存储（目前只支持），无论是用于语义搜索还是示例选择。

要导入此向量存储：
```python
from langchain.vectorstores import DeepLake
```


更详细的Deep Lake包装器操作步骤，请参考[此笔记本](/docs/modules/data_connection/vectorstores/integrations/deeplake.html)。
