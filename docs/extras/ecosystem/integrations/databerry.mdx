# Databerry

[Databerry](https://databerry.ai) 是一个开源的文档检索平台，帮助您将个人数据与大型语言模型连接起来。

## 安装和设置

我们需要注册 Databerry，创建一个数据存储，添加一些数据，并获取您的数据存储 API 端点 URL。我们需要 [API 密钥](https://docs.databerry.ai/api-reference/authentication)。

## 检索器

请参见[使用示例](/docs/modules/data_connection/retrievers/integrations/databerry.html)。

```python
from langchain.retrievers import DataberryRetriever
```
