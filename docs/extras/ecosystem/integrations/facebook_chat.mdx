# Facebook聊天

>[Messenger](https://en.wikipedia.org/wiki/Messenger_(software))是一款美国专有的即时通讯应用程序和平台，由`Meta Platforms`开发。最初于2008年开发为`Facebook Chat`，该公司在2010年改进了其消息服务。

## 安装和设置

首先，您需要安装`pandas` Python包。

```bash
pip install pandas
```
