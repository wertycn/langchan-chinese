# StochasticAI

本页面介绍了如何在LangChain中使用StochasticAI生态系统。
它分为两个部分：安装和设置，以及特定StochasticAI包装器的参考。

## 安装和设置
- 使用`pip install stochasticx`进行安装
- 获取一个StochasticAI API密钥，并将其设置为环境变量（`STOCHASTICAI_API_KEY`）

## 包装器

### LLM

存在一个StochasticAI LLM包装器，您可以通过以下方式访问它 
```python
from langchain.llms import StochasticAI
```
