# Google BigQuery

>[Google BigQuery](https://cloud.google.com/bigquery) 是一个无服务器且经济高效的企业级数据仓库，可在多个云平台上运行并根据数据规模进行扩展。
`BigQuery` 是 `Google Cloud Platform` 的一部分。

## 安装和设置

首先，您需要安装 `google-cloud-bigquery` Python 包。

```bash
pip install google-cloud-bigquery
```

## 文档加载器

请参考[用法示例](/docs/modules/data_connection/document_loaders/integrations/google_bigquery.html)。

```python
from langchain.document_loaders import BigQueryLoader
```
