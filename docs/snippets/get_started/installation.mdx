## 官方发布

要安装LangChain，请运行以下命令：

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
import CodeBlock from "@theme/CodeBlock";

<Tabs>
  <TabItem value="pip" label="Pip" default>
    <CodeBlock language="bash">pip install langchain</CodeBlock>
  </TabItem>
  <TabItem value="conda" label="Conda">
    <CodeBlock language="bash">conda install langchain -c conda-forge</CodeBlock>
  </TabItem>
</Tabs>

这将安装LangChain的最低要求。
LangChain的价值在于与各种模型提供商、数据存储等进行集成时的综合应用。
默认情况下，并未安装执行此操作所需的依赖项。
但是，有两种其他安装LangChain的方法可以引入这些依赖项。

要安装常见LLM提供商所需的模块，请运行：

```bash
pip install langchain[llms]
```

要安装所有集成所需的所有模块，请运行以下命令:

```bash
pip install langchain[all]
```

请注意，如果您正在使用`zsh`，在将方括号作为参数传递给命令时，需要对其进行引用，例如:

```bash
pip install 'langchain[all]'
```

## 从源代码安装

如果您想从源代码安装，可以通过克隆存储库并运行以下命令来进行安装：

```bash
pip install -e .
```
